## Contributing

If you have a favorite VS Code extension for learning that is not listed here, feel free to contribute! Follow these steps:

1. Fork the repository.
2. Create a new branch: `git checkout -b feature/new-extension`.
3. Add information about the new extension in the README.md file.
4. Create a pull request.

Your contribution is greatly appreciated!
